# TestFlight テスト案内

## What to Test

AIホストと音声で会話する流れを確認してください。

1. ホストタイプを選ぶ
2. 通話開始を押す
3. マイクボタンを押して話す
4. 文字起こしが表示される
5. AIホストの返答が表示され、音声で読み上げられる
6. 何度か会話を続ける
7. 会話終了を押す
8. 結果画面でログ、スコア、改善アドバイスを見る
9. 画面上部と下部にバナー広告が表示されるか確認する

## 確認してほしい点

- マイク許可と音声認識許可が自然に出るか
- 日本語の文字起こしが動くか
- 返答までの待ち時間が長すぎないか
- 読み上げ音声が聞き取りやすいか
- 速さ、高さ、音量の調整が効くか
- 会話終了後に結果画面が出るか
- 画面の文字が小さすぎないか
- ホストタイプごとの雰囲気が違うか
- 上下の広告が画面操作を邪魔しないか
- 広告表示後も通話画面が安定しているか

## 既知の制限

- 現在のAI返答はMockHostAIです。Gemma本体はまだ接続していません。
- Apple標準音声を使うため、専用音声AIほど感情表現は強くありません。
- 音声認識は端末や通信状態、Apple側の処理状況で精度が変わる場合があります。
- 広告は配信状況によって表示されない場合があります。

## フィードバック例

- 返答が速い / 遅い
- 声が聞き取りやすい / 聞き取りにくい
- ホストの口調が自然 / 不自然
- 画面が見やすい / 操作に迷った
- もう一度使いたいと思ったか
